## Features
- 🤖 Supports Claude, Gemini, OpenAI & Groq
- 🔑 BYOK — your API key stays in your browser only
- 📊 Prompt scoring (before vs after)
- 🟢 Diff view — see exactly what changed
- 💾 Local library with search, pin, notes & version history
- 📤 Export library as JSON or TXT
- 🎯 Templates per category

## Getting Started

1. Visit the live app: **https://promptly-ai-nine.vercel.app/**
2. Add your API key (free from any provider below)
3. Start optimizing!
